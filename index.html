Web Design Jaap

Sinds begin 2020 ben ik voor mijzelf begonnen in het veld van web development. Ik specialiseer mij hiermee in complete webpakketten met als basis een website, maar ook digital marketing en mailservices.

RaveShop

Op basis van een dropshipping principe verkoop ik op mijn eigen website RaveShop kleding en accessoires voor festivals, naar eigen ontwerp. Ik gebruik een leverancier die voor mij bedrukt, voorraad bijhoudt en levert.

StringTuner

Als resultaat van een profielwerkstuk heb ik een Play Store gepubliceerde app voor het stemmen van een ukulele. De app is gemaakt met JavaScript en gaat gepaard met een onderzoek naar frequenties en grondtonen.

Snackbar Lekkerrr Tafelreservering (CLE2)

Namens opdrachtgever Snackbar Lekkerrr te Ouddorp heb ik voor CLE2 een tafelreserveringssysteem gemaakt. Het systeem werkt volledig met een MySQL database en draait op PHP. Er is een admin systeem naast voor de opdrachtgever.

ReisVisie (CLE3)

Het groepsresultaat van CLE3 is ReisVisie, een platform wat reizen met het openbaar vervoer toegankelijk maakt voor mensen met een visuele beperking. Mijn specifieke pagina gebruikt de Google Maps API om op basis van je eigen locatie een route te genereren voor een station in de buurt.
